关于我们

由重庆大学本科2013级和2014级来自4个学院的8名同学共同创建。“多木”是英文Dormitory的谐音。多木创艺围绕大学生寝室生活服务，旨在开发一种全新的寝室整合式服务平台，更全面、直接、快捷地为大学生提供多样的寝室及周边服务。
